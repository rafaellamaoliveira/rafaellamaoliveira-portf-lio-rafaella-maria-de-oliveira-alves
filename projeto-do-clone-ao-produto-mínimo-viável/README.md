🏗️ Organização do Projeto
O SoftGrid.io é estruturado com base no uso do Neumorfismo, onde toda a interface segue um padrão visual consistente. A organização gira em torno de componentes reutilizáveis, como:

Botões: Elementos interativos com feedback tátil.

Cards (Painéis): Superfícies que se elevam suavemente do fundo.

Inputs (Campos de texto): Campos com aparência integrada à interface.

Containers (Blocos de conteúdo): Estruturas para organização de fluxo.

Todos esses elementos compartilham o mesmo estilo: superfícies suaves, sombras leves e uma aparência totalmente integrada ao fundo.

🎯 O que o projeto faz
O SoftGrid.io é uma plataforma que utiliza o design neumórfico para oferecer uma experiência visual mais moderna, simples e intuitiva, facilitando a interação do usuário com o sistema de forma orgânica e fluida.

✨ Funcionalidades:

As funcionalidades do sistema estão diretamente ligadas à interação visual:

Botões com efeito de “pressionado”: Feedback visual imediato ao clicar.

Cards com profundidade suave: Diferenciação de camadas sem poluição visual.

Inputs com aparência embutida: Integração visual perfeita ao interagir com os elementos.

Layout organizado e limpo: Sensação de que todos os elementos fazem parte da própria tela.

💻 Tecnologias:

O foco principal está no frontend, especialmente na construção da interface:

HTML: Estruturação da interface.

CSS: Principal tecnologia para criar o efeito neumórfico.

Uso de box-shadow (interno e externo).

Cores suaves e baixo contraste.

JavaScript: Interatividade, animações e gerenciamento de estados.

📚 Bibliotecas:

O projeto pode utilizar bibliotecas auxiliares, mantendo o diferencial do CSS customizado:

Frameworks de estilização: (ex: Tailwind CSS).

Bibliotecas de componentes UI.

Recursos nativos de CSS: Foco em design sem dependência excessiva de bibliotecas externas.
